## HTML API
  - `GET /` - loads React web app

## JSON API

###Users
  - `POST /api/users`
  - `PATCH /api/users`

###Session
  - `POST /api/session`
  - `DELETE /api/session`

###Sites
  - `GET /api/sites`
  - `POST /api/sites`
  - `GET /api/sites/:id`
  - `PATCH /api/sites/:id`
  - `DELETE /api/sites/:id`

###Reservations
  - `GET /api/reservations`
  - `POST /api/reservations`
  - `GET /api//reservations/:id`
  - `DELETE /api/reservations/:id`

###Reviews
  - `GET /api/reviews`
  - `POST /api/reviews`
  - `GET /api/reviews/:id`
  - `DELETE /api/reviews/:id`
